Timestamp Microservice API

Created by Daynil - Github Repo

Append a date to the end of the base URL in either:

  • Natural Language format: January 01, 2016

    • /January%2001,%202016

  • Unix timestamp (total seconds since 01/01/1970): 1451628000

    • /1451628000

Output for above queries:

{ "unix" : 1451628000, "natural" : "January 1, 2016" }
